## Features

- Register or login as an user.
- Update user account.
- Search products.
- Apply filters for searching products.
- Add products to cart.
- Change the required number for products.
- Payment with PayPal.
- Create order.
- Check order history.

##  NPM Packages Used

- React-redux
- Material UI
- Express
- Nodemon
- Mongoose
